# Python Practice & Projects

A collection of beginner Python scripts covering core concepts and mini projects.

## Files

| File | Description |
|------|-------------|
| `Variables & Data types.py` | Variables, data types, and type conversion |
| `Operators.py` | Arithmetic, comparison, and logical operators |
| `Python Syntax.py` | Basic Python syntax and structure |
| `Functions and modules.py` | Defining functions and importing modules |
| `Guess Game.py` | Number guessing game |
| `Rock Paper Game.py` | Rock, paper, scissors game |
| `Simple Calculator.py` | Basic calculator (add, subtract, multiply, divide) |
| `Password Generator.py` | Random password generator |
| `To Do List.py` | Command-line to-do list app |
| `tasks.txt` | Task data file used by To Do List |

## Topics Covered

- Variables and data types
- Operators
- Functions and modules
- Loops and conditionals
- String manipulation
- Random module
- File handling
- Mini projects

## Requirements

Python 3.x — no external libraries needed.

## How to Run

```bash
git clone https://github.com/rvr-likitha6/Python.git
cd Python
python "Guess Game.py"
```
